# dashword.org

A random dashword generator.

A dashword is a password that conists of:

- a capitalized word that is 12 characters or longer
- a single dash
- a 3 digit number

This yields a password that conforms to all modern complexity requirements:

- is at least 16 characters long
- contains uppercase letters
- contains numbers
- contains special characters (the dash)

To see it in action visit [dashword.org](https://dashword.org).

Note: don't use this for anything but temporary passwords. The dashwords will technically meet annoying complexity requirements but are not very complex or secure.
